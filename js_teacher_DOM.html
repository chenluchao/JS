DOM

DOM的概念

Document Object Model

DOM是W3C(万维网联盟)定义的一种标准。它定义了访问HTML文档元素的标准，它是关于如何获取、修改、添加和删除HTML元素的标准，即操作HTML元素。

DOM节点

根据W3C的DOM标准，HTML文档中的所有内容都是节点：

1、整个文档是一个文档节点
2、每个HTML元素是元素节点
3、HTML元素内的文本是文本节点
4、每个HTML属性是属性节点
5、注释是注释节点

整个文档可以被映射成一棵DOM树，树中的所有节点均可通过Javascript进行访问。所有HTML元素节点均可被修改，也可以创建或删除节点。

DOM树：

    html
    |-head
        |-title
            |-"文本"
    |-body
        |-h1
            |-"文本"
        |-div
            |-span
                |-"文本"
            |-a
                |-"文本"

DOM节点之间的关系：

1、父(parent)节点(node)
    父节点拥有任意数量的子节点。

2、子(child)节点(node)
    子节点只能拥有一个父节点。

3、同胞(sibling)节点(node)
    同级的子节点互为同胞。同胞是拥有相同父节点的节点。

4、根(root)节点(node)
    在节点树中，html节点被称为根。根节点没有父节点。

节点操作

childNodes

.childNodes获取父元素的所有子节点。

nodeType

.nodeType获取节点类型。

节点类型：
元素节点：1
文本节点：3
注释节点：8

nextSibling

.nextSibling获取指定节点之后紧跟的节点。

previousSibling

.previousSibling获取指定节点的前一个节点。

批量获取子元素

//通过childNodes获取所有子节点，再通过nodeType判断节点类型，筛选出所有元素节点
function getChildren(box){
    var nodes = box.childNodes;
    var result = [];
    for(var i=0; i<nodes.length; i++){
        if(nodes[i].nodeType == 1){
            result.push(nodes[i]);
        }
    }
    return result;
}

DOM元素节点的属性

tagName

ele.tagName表示元素节点的标记名称。

innerHTML

ele.innerHTML设置或者获取元素节点内的HTML内容。

outerHTML

ele.outerHTML设置或者获取元素节点的HTML内容，包含标记自身。

innerText

ele.innerText设置或者获取元素节点内的文本内容，HTML标记将被忽略。

id

ele.id设置或者获取元素节点的id名称。

title

ele.title设置或者获取元素节点的title属性值。

className

ele.className设置或者获取元素节点的类名。

//选项卡效果
var ebox = document.getElementById("box");
var eles = getChildren(ebox);
for(var i=0; i<ebox.length; i++){
    ebox[i].onclick=function(){
        for(var i=0; i<ebox.length; i++){
            ebox[i].className = "";
        }
        this.className = "active";
    }
}

style

style设置或者获取CSS内联样式属性。

offset*

offsetWidth只读属性，获取元素节点的可视宽度。

offsetHeight只读属性，获取元素节点的可视高度。

offsetLeft只读属性，获取元素节点的左偏移量。

offsetTop只读属性，获取元素节点的上偏移量。

获取元素样式的方法

ele.currentStyle获取元素的CSS样式表。IE8及以下使用，不支持复合样式。

window.getComputedStyle获取元素的CSS样式表。在高版本浏览器使用。

//获取元素样式表的兼容写法
function getStyle(ele, prop){
    return window.getComputedStyle?window.getComputedStyle(ele,null)[prop]:ele.currentStyle[prop]
}

DOM元素节点的方法

getElementById()

document.getElementById('id名')通过id名获取元素节点。document下的方法

getElementsByTagName()

ele.getElementsByTagName('标记名')通过标记名获取子元素节点集合。

getElementsByName()

ele.getElementsByName('name值')通过name值获取子元素节点集合。只有表单元素拥有name属性。

getElementsByClassName()

ele.getElementsByClassName('类名')通过类名获取子元素节点集合。IE8及以下不支持。

getAttribute()

ele.getAttribute('属性名')获取元素节点属性的属性值。可用于获取自定义属性的属性值。

setAttribute()

ele.setAttribute('属性名','属性值')设置元素节点属性的属性值。IE低版本不支持自定义属性。

removeAttribute()

ele.removeAttribute()移除元素节点的属性。某些低版本浏览器不支持。

DOM元素节点的方法

createElement()

document.createElement()创建一个元素节点。

createTextNode()

document.createElement()创建一个文本节点。

appendChild()

parent.appendChild(child)将child插入到parent的末端。

insertBefore()

parent.insertBefore(newChild, sibling)将newChild插入到sibling之前。

removeChild()

parent.removeChild(child)将child节点从parent节点中移除。

replaceChild()

parent.replaceChild(newChild, oldChild)用 newChild 节点替换 oldChild 节点。
